[font=verdana,geneva,sans-serif]ExtrabiomesXL is a mod for Minecraft. It adds new biomes that enhance game play and make exploring fun and interesting. ExtrabiomesXL is a collaborative open source project based on the MisterFiber's original idea and mod.[/font]

Prerequisite
Extrabiomes depends on Minecraft Forge, which is an API, or program layer, that provides many powerful features for modding Minecraft in ways that allow compatibility and interaction between mods. You can download the latest recommended version of Minecraft Forge [url="http://jenkins.minecraftforge.net:7070/job/Forge/forge-recommended/"]here[/url].

Downloads
Extrabiomes is delivered as one "universal" download that can be used for both SSP and SMP.
